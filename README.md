# РобоРейнджеры

Сайт кружка робототехники для детей 7-15 лет.

## Файлы

- `index.html` - структура сайта
- `styles.css` - оформление и адаптивность
- `script.js` - меню, вкладки курсов и демонстрационная форма

## Как открыть

Откройте `index.html` в браузере.

Для удобного редактирования можно использовать Visual Studio Code и расширение Live Server.

## Что заменить

1. Телефон, адрес и Telegram.
2. Стоимость занятий.
3. Названия курсов.
4. Тексты проектов.
5. Подключение формы к Telegram-боту, почте или CRM.
