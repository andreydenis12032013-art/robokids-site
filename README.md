# РобоСтарт — сайт кружка робототехники

Готовая статическая версия сайта для обучения детей робототехнике.

## Файлы

- `index.html` — структура сайта
- `styles.css` — оформление и адаптивность
- `script.js` — меню, переключение курсов и демонстрационная форма

## Как запустить

Откройте `index.html` в браузере.

## Что заменить под себя

1. Название кружка: `РобоСтарт`
2. Телефон, адрес и Telegram в блоке записи
3. Стоимость занятий
4. Возрастные группы и программу
5. Подключение формы:
   - Telegram-бот
   - email
   - Google Forms
   - CRM
